You may specified the IP, mask or the host-counts in the wrong format!

Correct formats are:

IP Address: X.X.X.X where 'X' is a number between 0-255
Subnet mask: X.X.X.X where 'X' is a number between 0-255, or /X where 'X' is a number between 0-31
Host counts: r,r,r,r...., where 'r's are custom numbers e.g 15,34,12,1,4
